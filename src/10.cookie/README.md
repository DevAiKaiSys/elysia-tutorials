# 🍪 Cookie Management in Elysia

Learn how to handle cookies effectively with Elysia's reactive cookie API. This tutorial demonstrates:
- Accessing and modifying cookies
- Reactive cookie updates
- Cookie type validation with `t.Object`
- Configuring cookie options (e.g., `httpOnly`)

## How to Run

Execute the following command to start the development server:

```bash
bun run --watch src/10.cookie/index.ts
```

Navigate to [http://localhost:3000](http://localhost:3000) to see the visit counter in action.
